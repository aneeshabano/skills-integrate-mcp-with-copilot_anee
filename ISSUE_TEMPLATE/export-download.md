# Feature: Export/Download (user download of generated docs)

## Summary
Allow users to export and download generated documents (DOCX/PDF) directly from the application interface.

## Details
- Provide a clear download button after document generation.
- Ensure files are named meaningfully (e.g., activity-participant-date.docx/pdf).
- Support multiple formats (DOCX, PDF).

## Acceptance Criteria
- Users can download generated documents in their preferred format.
- Downloaded files are named appropriately.
- Download process is reliable and user-friendly.
