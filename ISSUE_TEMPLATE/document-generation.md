# Feature: Document Generation (docx/pdf export)

## Summary
Enable users to generate documents (such as activity participation certificates or reports) in DOCX and PDF formats directly from the application.

## Details
- Integrate a document generation library (e.g., python-docx for DOCX, abiword or similar for PDF export).
- Allow users to select an activity and generate a document with relevant details.
- Support for exporting the generated document as a downloadable file.

## Acceptance Criteria
- Users can generate a DOCX document for an activity.
- Users can export the document as a PDF.
- Download link is provided after generation.

Inspired by features in the Cover Letter Generator project.
