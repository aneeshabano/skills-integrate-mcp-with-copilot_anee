# Feature: Template Customization (user-editable templates)

## Summary
Allow users (admins/teachers) to customize the templates used for document generation, so that generated documents can be tailored to specific needs or branding.

## Details
- Provide a UI for editing document templates (e.g., placeholders for activity name, date, participant, etc.).
- Store templates in a user-editable format (e.g., DOCX, HTML, or a simple markup).
- Support previewing changes before saving.

## Acceptance Criteria
- Admins/teachers can edit and save document templates.
- Changes are reflected in newly generated documents.
- Template editing is user-friendly and secure.
